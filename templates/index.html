About This Tool

This tool automatically scans websites for WCAG 2.1 Level AA accessibility compliance issues.

What we check:
  • Color contrast
  • Keyboard navigation
  • Screen reader compatibility
  • Form labels and errors
  • Image alt text
  • Heading structure
Tips
  • Start with a smaller number of pages for faster results
  • Use path restriction for focused scans of specific sections
  • Disable screenshots for faster scanning
  • Check the history page to view previous scans
